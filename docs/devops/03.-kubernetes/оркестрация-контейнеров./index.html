    Подключение к GitLab#

    .dockerconfigjson Создание и проверка токена ссылка

    • With Git, when using HTTP Basic Authentication, use:
      • Any non-blank value as a username.
      • The project access token as the passwor Для кодирования токена нужно использовать echo -n <tocken-name>:<tocken> | base64, для кодирования файла аутентификация в secret - cat file | base64 -w 0 Для сборки нужно
    • создать .gitlab-ci.yml
      • скачать и установить kubectl
      • вызывать kubectl apply -f ….
    • создать файловую (file) переменную KUBECONFIG c содержимым .kube/config.
