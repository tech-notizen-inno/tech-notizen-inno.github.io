    Vagrant#

    Поддерживает любой гипервизор, управление собственным командным интерфейсом Vagrant. Например, мы можем использовать API VirtualBox, и все виртуалки будут видны в UI. Самое логичное использование — локальная разработка, либо тестирование. В основе Vagrant лежат box’ы. Box’ы ты можешь делать сам или качать с сайта Vagrant’а.

    • Инструкция по CLI

    Последовательность использования#

    1. Установка VirtualBox, которым будет управлять Vagrant sudo apt-get update -y sudo apt-get install virtualbox virtualbox-ext-pack -y
    2. Установка Vargrant под Ubuntu $ curl -fsSL https://apt.releases.hashicorp.com/gpg | sudo apt-key add - $ sudo apt-add-repository "deb [arch=amd64] https://apt.releases.hashicorp.com $(lsb_release -cs) main" $ sudo apt-get update && sudo apt-get install vagrant
    3. Создать папку и проинициализировать виртуалку vagrant init generic/ubuntu2010
    4. Создать скрипт provisioning (что закинуть на ВМ) echo 'deb https://nginx.org/packages/ubuntu/ focal nginx' > /etc/apt/sources.list.d/nginx.list echo 'deb-src https://nginx.org/packages/ubuntu/ focal nginx' >> /etc/apt/sources.list.d/nginx.list apt-key adv –keyserver keyserver.ubuntu.com –recv-keys ABF5BD827BD9BF62 apt-get update -y pngt-get install nginx -y systemctl enable nginx systemctl start nginx
    5. Добавить в конфигурацию ссылку на provisioning config.vm.provision :shell, path: "bootstrap.sh"
    6. Запустить VM vagrant up --provider=virtualbox или vagrant reload --provision
    7. Статус VM vagrant status
