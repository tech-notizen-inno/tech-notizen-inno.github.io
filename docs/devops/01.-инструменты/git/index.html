    Команды Git#

    • git config --global credential.helper 'cache --timeout=3600 - кэширование пароля
    • git branch -d feature - удаление ветки локальном репозитории
    • git push origin :feature - удаление ветки удалённом репозитории
    • git log feature --oneline - посмотреть лог в виде 1 строки
    • git branch -r - посмотреть ветки на удаленном репозитории
    • git checkout -b my-feature origin/my-feature - переключиться на ветку удаленного репозитория
