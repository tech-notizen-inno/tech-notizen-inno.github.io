    Linux. Бэкапирование

    • Удаление бэкапа с помощью cron и find

    RSync#

    rsync -azvh <user1>@<host>:/path_to_file/file.ext <backup_user>@<backup_host>:/path_to_file/ > /var/log/backup.log

    Пример

    rsync -azvh <u1>@<sausage>:/home/jarservice/sausage-store.jar <u2>@<backup>:/mnt/sausage_backup/$(date +"%m-%d-%Y")/ > /var/log/sausage-store.log

    $(date +"%Y-%m-%d_%H-%M-%S") - вывод даты с точностью до секунд

    Удаление бэкапа с помощью cron и find#

    Пример

    `0 3 * * * find /mnt/sausage_backup/ -name "sausage-store.jar" -type f -mtime +14 -delete`
