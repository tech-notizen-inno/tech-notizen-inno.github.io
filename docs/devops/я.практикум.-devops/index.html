    Стратегия деплоя#

    Стратегия деплоя — это последовательность действий для изменения или обновления запущенного приложения.

    Стратегия Описание
    Recreate Пересоздание — это когда старая версия просто выключается, и после этого запускается новая.
    Rolling Update ![[PI-20241116205221.png]]
    Blue/green ![[PI-20241116205308.png]]
    Canary При выкатке на первую группу, составляющую, например, 2% от всего числа пользователей, не будет выявлено ошибок, то размер группы увеличивается инкрементально (например: 2, 10, 25, 75, 100%).
