    Gradle#

    Шаги по генерации кода из OpenAPI spec
    1. Подключить plugin к нужному модулю. Можно использовать alias
    **2. Задать путь к спецификации **
    **3. Установить директорию для исходников (куда будет сгенерены файлы) **
    еще пример
    **4. Внести таску для генерации **
    **5. Установить зависимость компиляции от генерации **
    Параметры генерации Ссылка Описание параметра
    generatorName язык, который будет использоваться для генерации.
    inputSpec путь к файлу спецификации.
    additionalProperties дополнительные настройки. Список
    outputDir Настраиваем директорию вывода для плагина
    typeMappings Маппинг типов из spec на Kotlin
    logToStderr Выводить сообщения в stderror
    verbose Включить режим максимального вывода сообщений
    importMappings Использование маппера для типов
