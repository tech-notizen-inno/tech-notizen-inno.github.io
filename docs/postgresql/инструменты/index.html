    psql#

    psql "host=<db host> port=6432 sslmode=verify-full dbname=<db name> user=devops target_session_attrs=read-write" - подключение к БД локальным клиентом

    Основные команды

    • \dt - показать таблицы
    • \df - показать функции
    • \d product - просмотреть структуру таблицы product
    • \timing - запуск тайминга запросов
    • \dF, \dFp, \dFp+ - List of text search configurations, List of text search parsers, Token types for parser (отображение опций полнотекстового поиска)

    pgbench#

    • pgbench -n -h 127.0.0.1 -d postgres -T 300 2> /dev/null - запуск утилиты с выводом stderr в /dev/null

    Чистка коннектов к БД#

    • Подключениия храняться в таблице pg_stat_activity. Посмотреть подключения SELECT * from pg_stat_activity WHERE datname='std-028-50'
    • Комманда DISCONNECT
    • SELECT pg_terminate_backend(pid) FROM pg_stat_activity WHERE datname = ‘std-028-50’ AND state = ‘idle’ AND pid <> pg_backend_pid();

    Создание dump#

    • создание backup - pg_dump postgresql://$PSQL_USER:$PSQL_PASSWORD@$PSQL_HOST:$PSQL_PORT/$PSQL_DBNAME -j 8 -Fd -f /tmp/dump.dir
    • восстановление backup - pg_restore -d "postgresql://$PSQL_USER:$PSQL_PASSWORD@$PSQL_HOST:$PSQL_PORT/$PSQL_DBNAME" --format=d /tmp/dump.dir/

    PgBouncer#

    1. установить (apt install) pgbouncer и запусить system.d unit
    2. сконфигурировать роль
    set password_encryption = 'scram-sha-256'; 
create role pgbouncer; 
alter role pgbouncer login password 'postgres';
    1. генерируем файл userlist.txt, который PgBouncer будет использовать для авторизации в Postgres: psql -d postgres -U postgres -c "SELECT concat('\"', usename, '\" \"', passwd, '\"') FROM pg_shadow where usename in ('postgres','pgbouncer')" -o <DIR>/userlist.txt + переписать в папку /etc/pgbouncer
    2. конфигурируем /etc/pgbouncer/pgbouncer.ini. пример
    3. рестрат сервиса
    sudo systemctl restart pgbouncer
sudo systemctl status pgbouncer

    Команды pgbouncer#

    • show pool - показать настроенные пулы соединений
    • show help - показат соединения
